Tableau Commandes Git
Commande Description
Git config On l'utilise pour configurer les préférences de l'utilisateur
Git init Cette commande est utilisée pour créer un nouveau dépôt GIT
Git add Elle peut être utilisée pour ajouter des fichiers à l'index
Clone git Elle est utilisée pour la vérification des dépôts
Git commit Elle permet de valider les modifications apportées au HEAD
Git status Elle affiche la liste des fichiers modifiés ainsi que les fichiers qui doivent encore être ajoutés ou validés
Git push Un simple push envoie les modifications locales apportées à la branche principale associée
Git checkout Elle peut être utilisée pour créer des branches ou pour basculer entre elles
Git remote permet à un utilisateur de se connecter à un dépôt distant
Branche git Elle peut être utilisée pour répertorier, créer ou supprimer des branches
Git pull Pour fusionner toutes les modifications présentes sur le dépôt distant dans le répertoire de travail local, la commande pull est utilisée
Git merge Elle est utilisée pour fusionner une branche dans la branche active
Git tag Le marquage est utilisé pour marquer des commits spécifiques avec des poignées simples
Git log L'exécution de cette commande génère le log d'une branche
Git reset Pour réinitialiser l'index et le répertoire de travail à l'état du dernier commit.
Git rm Elle peut être utilisé pour supprimer des fichiers de l'index et du répertoire de travail
Git ls-tree Pour afficher un fichier arborescent avec le nom et le mode de chaque élément
Git show Pour afficher des informations sur tout fichier git
Git fetch Elle permet à un utilisateur d’extraire tous les fichiers du dépôt distant qui ne sont pas actuellement dans le répertoire de travail local
Git stash Elle aide à enregistrer les changements qui ne doivent pas être commit immédiatement
Git cat-file Elle affiche le type d'un fichier
Git grep Elle permet à un utilisateur de rechercher dans les arbres de contenu des expressions et / ou des mots
Gitk Interface graphique du dépôt local
Git rebase Elle est utilisée pour la réapplication des commits sur une autre branche
Git gc Optimise le dépôt en supprimant les fichiers inutiles et les optimiser
Git archive Elle permet à un utilisateur de créer un fichier zip ou tar contenant les composants d'un arbre du dépôt
Git prune Les fichiers qui n'ont pas de pointeurs entrants seront supprimés
Git instaweb Elle permet à un serveur Web peut être exécuté par interface avec le dépôt local
Git fsck Effectue une vérification d'intégrité du système de fichiers git
